## 1. 部署到公网上 ✅️

在 Namesilo 购买了 smlcoke.com 域名，并且借助 github pages 成功部署了这个网站，同时还启用了 Cloudflare 的 CDN 服务。

github pages 的部署流程：
1. 在 github 上创建一个新的 repository，命名为 `smlcoke.github.io`。
2. 将 MkDocs 生成的静态文件推送到这个 repository 的 `main` 分支。
3. 在 repository 的设置中启用 GitHub Pages，选择 `main` 分支作为发布源。

<!-- more -->

全流程所花时间只有2h（by Google AI Studio Gemini 3.1 Pro Preview），但是从购买域名到 Cloudflare 配置一系列操作全靠 Gemini 知道，自己一路懵逼，不知道这些操作是什么意思。所幸聊天记录尚在，以后有时间看看吧。
