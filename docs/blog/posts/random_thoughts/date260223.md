# 当前进度

## 1. Rust 加密工具 ✅️

Rust 的加密工具 `ciphery` 已经完成了基本的功能开发（虽然目前只支持 `Caesar` 加密）。
成功 Release，这个比较简单。
然后再利用 WASM 技术将其部署到了本网站，期间发现 Rust 的 `#[cfg(target = 'wasm32')]` 条件编译功能非常强大，可以让我们**在同一份代码中针对不同的目标平台编写不同的实现**，非常方便。缓过神来才发现这一点跟之前的自动编写测试 `#[cfg(test)]` 如出一辙。
