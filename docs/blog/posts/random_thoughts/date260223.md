# 当前进度

## 1. Rust 加密工具 ✅️

<!-- more -->
1. Rust 的加密工具 `ciphery` 已经完成了基本的功能开发（虽然目前只支持 `Caesar` 加密）。
2. 成功 Release，这个比较简单。
3. 然后再利用 WASM 技术将其部署到了本网站，期间发现 Rust 的 
   ```rust
   #[cfg(target_arch = "wasm32")]
   #[wasm_bindgen]
   ```
   条件编译功能非常强大，可以**在同一份代码中针对不同的目标平台编写不同的实现**，非常方便。缓过神来才发现这一点跟之前的自动编写测试 `#[cfg(test)]` 如出一辙。
4. 测试好后利用 `wasm-pack build --target web` 命令将 Rust 代码编译成 WebAssembly 模块。
5. 将 `pkg` 文件夹中的文件防止 web 项目，加密工具主页由 crypto.md 提供。内置一个简单的 UI。
